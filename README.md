# AGBCN5 - Công cụ Số Agribank CN 5

Ứng dụng PWA đẹp, mượt, đầy đủ tính năng cho nhân viên Agribank Chi nhánh 5.

## Cách sử dụng ngay trên điện thoại

1. Mở file `index.html` bằng trình duyệt Chrome / Safari trên điện thoại.
2. Nhấn nút **Cài đặt** (hoặc "Thêm vào màn hình chính").
3. Ứng dụng sẽ hoạt động như app thật, có thể dùng offline sau lần đầu.

## Tạo icon PWA (bắt buộc cho trải nghiệm đẹp)

Mở app → vào tab **Khác** → nhấn nút **"Tạo & Tải Icon PWA"** → lưu 2 file PNG vào thư mục `assets/icons/`.

Sau đó reload app và cài lại PWA để có icon đẹp trên màn hình chính.

## Tính năng chính

- **VietQR**: Tạo mã QR chuyển khoản chuẩn VietQR (scannable mọi ngân hàng VN)
- **QR Map Định Vị**: Lấy GPS + bản đồ Leaflet + tạo QR chứa tọa độ + thông tin khách hàng
- **Lãi suất**: Bảng lãi suất huy động Agribank CN5 (cập nhật giả lập)
- **Máy tính lãi**: Tính lãi tiền gửi & tiền vay chính xác (công thức 365 ngày)
- **Card Visit Số**: Tạo QR danh thiếp điện tử (vCard)
